游戏类型：
- 2D 俯视角几何射击动作游戏
- 核心体验是：走位、瞄准、切武器、冲刺、读敌人节奏、看强反馈特效

MVP 核心循环：
- 玩家进入一个战斗场
- 使用 WASD 移动、鼠标瞄准和射击
- 通过切换不同武器应对不同敌人
- 通过冲刺规避伤害或调整位置
- 敌人持续刷新并施加不同节奏压力
- 玩家存活并击败一名 mini boss，算作一轮 MVP 闭环成立

玩家输入：
- WASD：移动
- 鼠标：瞄准
- 左键：主攻击
- 右键：副模式或辅助瞄准/替代攻击
- 空格：冲刺
- 1/2/3：切换三种武器
- 4 可以预留，不一定实现

首批武器建议：
1. 机枪：连续射击，命中反馈密集，适合基础清怪
2. 榴弹：低频高爆发，范围伤害，强调爆炸与震动
3. 狙击：高精度高伤害，穿透或弱点加成，强调击中时刻

首批敌人建议：
1. 近战追击怪：简单追击玩家
2. 远程节奏怪：停顿瞄准后发射弹体
3. 冲锋怪或精英怪：有明显前摇与突进

mini boss 要求：
- 有明确阶段感或至少两种技能
- 有可读前摇
- 能测试玩家是否会用冲刺和切武器
- 暂时不要做太复杂机制

MVP 的成功标准：
- 移动和冲刺手感明显不同于普通网页 demo
- 三种武器有足够差异
- 命中和爆炸反馈让人觉得“爽”
- 敌人攻击节奏可读
- 画面简洁但动态丰富

这个项目的视觉重点不是复杂美术，而是“极简几何 + 强反馈特效”。

请以以下视觉原则实现：
1. 角色、敌人、子弹、爆炸尽量使用几何图形和程序化绘制
2. 颜色数量要克制，整体现代、简洁、锐利
3. 通过以下手段制造爽感：
   - 命中闪白
   - 受击闪烁
   - 击杀碎裂
   - 粒子飞散
   - 爆炸扩散环
   - 子弹拖尾
   - 冲刺残影
   - 屏幕轻微震动
   - 武器射击时的后坐/镜头推拉感
4. 特效必须优先服务于反馈，而不是单纯热闹
5. 所有特效都尽量参数化，方便我后续调节强弱
6. 在不依赖复杂贴图的前提下，尽可能做出“干净但有冲击力”的观感

视觉风格关键词：
- 极简
- 现代
- 几何
- 霓虹但克制
- 高对比
- 快速
- 清脆
- 有节奏的冲击感

请优先完成以下反馈层：
- 玩家移动拖尾
- 冲刺残影
- 子弹轨迹
- 命中火花/碎片
- 榴弹爆炸波纹
- 狙击命中时的强反馈
- 敌人死亡几何碎裂
- HUD 的轻微动态反馈
